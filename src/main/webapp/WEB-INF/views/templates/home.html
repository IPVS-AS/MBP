Welcome to the MBP!

  • keyboard_arrow_down

We are happy that you have chosen the MBP to manage your IoT devices!


Let's start to set up your IoT environment in only six steps:

1. devices  Register the IoT devices of your environment. Optionally, you can register vpn_key  Key Pairs to be used by the MBP to access your IoT devices remotely.

2. description  Create extraction/control operators for your sensors and actuators. Control operators are only required if there are actuators in your environment.

3. settings_remote  Register sensors and wb_incandescent actuators that are connected to your devices, by selecting a target device and an operator.

4. file_download Install operators for your sensors and actuators on target devices.

5. play_arrow Start your sensors and actuators.

6. call_split Define rules that make your environment smart and allow it to react to certain conditions.


As an alternative, you can use the view_quilt Environment Modeling Tool to set up your IoT environment at once.
